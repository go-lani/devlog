objectKit Examples

🧩

isEmptyObject - 빈 객체 체크

주어진 객체가 빈 객체인지 확인합니다.

objectKit.isEmptyObject({})
결과:
빈 객체: true
objectKit.isEmptyObject({ a: 1 })
결과:
값이 있는 객체: false
🔧

getCompleteObject - 완전한 객체 반환

값이 undefined이거나 null인 키를 제거하여 완전한 객체를 반환합니다.

objectKit.getCompleteObject({ a: 1, b: null, c: 'test', d: undefined })
결과:
{
  "a": 1,
  "c": "test"
}
🔍

getObjectKeyByValue - 값으로 키 찾기

객체에서 특정 값에 해당하는 키를 찾습니다.

objectKit.getObjectKeyByValue({ red: '#ff0000', blue: '#0000ff' }, '#ff0000')
결과:
red
