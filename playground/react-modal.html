Props 정의

NameTypeRequiredDescription
component(closeModal: () => Promise<void>) => JSX.ElementRequired모달 안에 렌더링될 컴포넌트
isOpenbooleanRequired모달 열림/닫힘 상태
onClose() => unknownOptional모달 닫힐 때 호출되는 콜백
namestringOptional모달 식별자 (기본값: 'modal')
dimstringOptional배경 딤 색상 (예: 'rgba(0, 0, 0, 0.8)')
centerModebooleanOptional모달을 화면 중앙에 배치할지 여부 (기본값: false)
animation{ className?: string; duration: number }Optional애니메이션 설정
containerPaddingstringOptional모달 컨테이너 패딩
isUnlockScrollbooleanOptional스크롤 잠금 해제 여부 (기본값: false)
disabledOutsideClosebooleanOptional외부 클릭으로 닫기 비활성화 여부 (기본값: false)

Usage

1import { Modal } from '@lani.ground/react-modal';
2import '@lani.ground/react-modal/css';
3
4const [isOpen, setIsOpen] = useState(false);
5
6<button type="button" onClick={() => setIsOpen(true)}>
7  Open Modal
8</button>
9
10<Modal
11  name="preview"
12  component: (closeModal: () => Promise<void>) => (
13    <div>
14      <div>PopupContent</div>
15      <button type="button" onClick={closeModal}>Close</button>
16    </div>
17  ),
18  onClose={() => setIsOpen(false)}
19  animation={{
20    className: 'sample',
21    duration: 300,
22  }}
23  dim="rgba(0, 0, 0, 0.8)"
24  isOpen={isOpen}
25  containerPadding="20px"
26  centerMode
27  disabledOutsideClose
28/>

Modal Examples

🪟

Direct Modal

페이지 렌더링시 모달을 즉시 노출 시킬 수 있습니다.

주요 기능

  • 중앙 정렬
  • 딤 배경
  • 스크롤 잠금 해제
  • 애니메이션
🪟

Basic Modal

기본적인 모달 컴포넌트입니다. 중앙 정렬과 딤 배경이 적용되며, 스크롤 잠금 기능을 제공합니다.

주요 기능

  • 중앙 정렬
  • 딤 배경
  • 스크롤 잠금 해제
  • 애니메이션
🎲

Conditional Modal

조건부 렌더링이 가능한 모달입니다. 랜덤하게 유/무효 팝업을 보여줍니다.

주요 기능

  • 조건부 렌더링
  • 랜덤 상태
  • 자동 닫기
  • 상태 표시
⚙️

Advanced Modal

고급 기능이 포함된 모달입니다. 외부 클릭 비활성화, 컨테이너 패딩, 복잡한 컴포넌트를 포함합니다.

주요 기능

  • 외부 클릭 비활성화
  • 컨테이너 패딩
  • 복잡한 UI
  • 중앙 정렬
